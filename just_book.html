A collection of some of my projects

Pill Tracker

Written in Ruby and using Sinatra to complete the MVC model and using PostGreSQL as it's database.

This project was to allow someone to save pill details and then register when they took their pill so they wouldn't forget when they took them.

Portfolio Tracker ShareWatch

Using Node.js for a backend to a MongoDB and Vue.js for the front end

This project used a public API to look up stocks in the NYSE and then be able to register the purchase of the stock including the date and price

Mobile Restaurant Booking System

Using Java and React Native to create a mobile app that the staff can use to save repeat customers and book them into tables
